### BookVault

**Cloud-Native Book Catalog API Platform**

BookVault is a containerized API platform for searching and retrieving book data, built with an AWS-first architecture and a strong focus on automation, reliability, and operational clarity.

**Highlights:**
- Built a containerized API with automated endpoint testing
- Designed the service for deployment on AWS using Infrastructure as Code
- Structured the project around private networking, centralized logging, and CI/CD readiness

**Tech Stack:** AWS, ECS Fargate, ECR, CloudWatch, GitHub Actions, Terraform, Docker, Python, FastAPI

### BookCluster

**Kubernetes Platform for Book Microservices**

BookCluster is a Kubernetes-based platform for running a set of lightweight book-related microservices, with a focus on orchestration, networking, deployment automation, and cluster observability.

**Highlights:**
- Orchestrates multiple services including Redis using Kubernetes Deployments and Services
- Uses production-style platform features such as ConfigMaps and Horizontal Pod Autoscaling
- Includes observability components with Prometheus and Grafana for service monitoring

**Tech Stack:** Kubernetes, Docker, YAML, Redis, Prometheus, Grafana, Ingress, GitHub Actions
